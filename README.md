CookBook - Personal Recipe Manager

CookBook is a full-stack web application that allows users to discover, create, and manage their favorite recipes. It features secure user authentication, image hosting, and a database to persist user creations.

Features

- User Authentication: Secure login and signup using Auth0.
- Recipe Management: Create, Read, Update, and Delete (CRUD) your own recipes.
- Image Uploads: Upload recipe photos directly via Cloudinary.
- Search & Filter: Search for recipes by name or filter by ingredients.
- Favorites: Save recipes to your personal favorites list.
- Responsive Design: Works on desktop and mobile.

Tech Stack :

Frontend:
- React (Vite)

Backend & Services:
- Netlify Functions: Serverless backend logic.
- MongoDB Atlas: NoSQL database for storing recipes and user data.
- Auth0: Identity management.
- Cloudinary: Cloud storage for images.
